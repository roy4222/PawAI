| 機位 | 位置 | 用途 |

| 步驟 | 動作 | 預期結果 | 時長 | 備註 |

| **4. GPU 推論** | YOLO + DA3 處理 | 回傳 JSON 結果 | 2秒 | 可加字幕顯示 JSON |
| **5. Go2 移動** | 朝椅子方向前進 | 平穩移動，無碰撞 | 10秒 | 主鏡頭追蹤 |

| **8. 執行動作** | `go2_perform_action('Hello')` | Go2 揮手打招呼 | 5秒 | 結尾亮點 |

## 🎬 場景 2：多物件環境測試（技術展示）

### 目標
展示系統能在複雜環境中「精準識別目標物」，不被干擾物混淆

### 場地佈置

```
   [椅子]    [水瓶]    [紙箱]
     ↑         ↑         ↑
    2m        2.5m       3m

            [Go2]
            起點
```

**道具位置：**
- Go2 起點：客廳中央
- 椅子：左前方 2m
- 水瓶：正前方 2.5m（放在桌上）
- 紙箱：右前方 3m

### 執行腳本

| 步驟 | 動作 | 預期結果 | 時長 |
|------|------|----------|------|
| **1. 下達指令** | 「幫我找水瓶」 | 螢幕顯示指令 | 2秒 |
| **2. 環顧掃描** | Go2 左轉 → 中 → 右轉 | 擷取 3 張圖片 | 8秒 |
| **3. AI 分析** | LLM：「偵測到椅子、水瓶、紙箱」 | 螢幕顯示物件清單 | 3秒 |
| **4. 目標鎖定** | LLM：「鎖定目標：水瓶」 | 高亮顯示 JSON | 2秒 |
| **5. 移動** | 前進至水瓶位置 | 平穩移動 | 10秒 |
| **6. 完成** | 停在水瓶前 | 精準定位 | 2秒 |

**總時長：** 約 27 秒

### 錄製要點

✅ **成功標準：**
- 正確識別水瓶（不混淆椅子/紙箱）
- 移動路徑直線、無繞路

📝 **字幕建議：**
```

00:03 - 環顧四周，掃描環境...
```

## 📝 錄製注意事項

### 技術風險管理

| 風險 | 預防措施 | 應對方案 |
|------|---------|---------|
| **WebRTC 斷線** | 提前重啟 Go2 + VM | 立即重連，等待 30 秒 |
| **LLM 回應錯誤** | 使用已驗證的 Prompt | 重錄該段落 |
| **GPU Server 延遲** | 錄製前測試 `/find_object` | 確認網路穩定，必要時重啟 |
| **Go2 電量不足** | 每場景後檢查電量 | 準備充電器，中場休息充電 |

### 安全注意事項

⚠️ **移動安全：**
- 確保 Go2 移動範圍內無障礙物
- 設定 Safety Layer 最大速度（0.3 m/s）
- 隨時準備緊急停止（ESC 鍵 / 手把）

⚠️ **設備安全：**
- 三腳架固定穩固（避免傾倒）
- Go2 避免跌落、碰撞
- 錄影設備避免遮擋 Go2 視線
