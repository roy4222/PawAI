## 跑法

到 `http://localhost:3001/studio`（或語音對麥克風）：
1. 一次跑一題，記 ✅ / ⚠️ / ❌ + 觀察
2. 觀察 brain trace（另開 terminal）：
   ```bash
   ssh jetson-nano 'source /opt/ros/humble/setup.zsh && source ~/elder_and_dog/install/setup.zsh \
     && ros2 topic echo /brain/conversation_trace --field data' \
     | jq -c 'select(.stage == "mode_classify" or .stage == "gesture_gate" or .stage == "verifier")'
   ```
3. 對話間隔 ≥ 35s 才測手勢（不然 N6 gate 會擋）
4. 中段 reset：Studio 「新對話」按鈕清 context，跑 D 系列前重置
