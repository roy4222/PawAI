## ⚠️ Phase 2：API 服務化 (11/27 - 12/03)

**目標：** 把 AI 模型包裝成一個「網站服務 (Web API)」，讓 A 組可以呼叫。
*這階段你們會跟 B 組緊密合作，把程式放上伺服器。*

- **撰寫 API Server (`inference_server.py`)**
    - **框架**：使用 **FastAPI** (Python 最快的 API 框架)。
    - **功能**：
        1. 接收 `HTTP POST` 上傳的圖片。
        2. 丟進 YOLO 模型運算。
        3. 回傳 JSON 格式結果。
    - **JSON 格式約定 (Contract)**：
        
        ```json
        {
            "detected": true,
            "object": "cup",
            "bbox": [100, 200, 50, 80],  // x, y, w, h
            "center": [125, 240]         // A 組最需要這個！
        }
        
        ```
        
- **部署到 GPU 伺服器**
    - [ ]  把程式碼傳給 B 組。
    - [ ]  確認在 RTX 8000 上跑得起來，而且速度要快 (< 0.5秒)。
