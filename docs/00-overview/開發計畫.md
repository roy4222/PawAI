## 1. 開發計畫總覽

### 1.1 架構轉型說明

| 項目 | 舊架構 (v3.x) | 新架構 (v4.0) | 變更理由 |
|------|--------------|--------------|---------| 
| **專題名稱** | 老人與狗 | **PawAI** | 功能導向，避免被質疑受眾 |
| **視覺處理** | YOLO-World + DA3 (雲端) | **coco_detector (本地) + 深度攝影機** | 降低延遲，Edge AI |

| **連接方式** | WebRTC (WiFi) | **CycloneDDS (有線)** | 延遲 <1ms |
| **AI 架構** | 單一雲端 | **雙層：本地 + GPU Server** | 即時 + 高階推理分離 |
| **GPU 用途** | YOLO 推論 | **本地 LLM + VLM + STT** | 發揮 RTX 6000 算力 |

## 2. 雙層系統架構

### 2.1 架構圖

```
┌─────────────────────────────────────────────────────────────────┐
│                學校 GPU Server (RTX 6000 Blackwell)              │
│  ┌────────────────────────────────────────────────────────┐     │
│  │          慢系統 (System 2) - 高階推理 1-5s             │     │
│  ├────────────────────────────────────────────────────────┤     │
│  │  Qwen2.5-72B (語意)  │  LLaVA-34B (VLM)               │     │
│  │  Whisper V3 (STT)    │  SAM 2 (追蹤)                  │     │
│  └────────────────────────────────────────────────────────┘     │
└─────────────────────────────┬───────────────────────────────────┘
                              │ HTTP API
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│                    Mac VM (Ubuntu 22.04) - 大腦                  │
│  ┌────────────────────────────────────────────────────────┐     │
│  │         快系統 (System 1) - 即時控制 <200ms            │     │
│  ├────────────────────────────────────────────────────────┤     │
│  │  Nav2 + SLAM       │  coco_detector                   │     │
│  │  深度攝影機        │  TTS (ElevenLabs)                │     │
│  │  Safety Layer      │  Skills 模組                     │     │
│  │  MCP 協定          │  rosbridge                       │     │
│  └────────────────────────────────────────────────────────┘     │
└─────────────────────────────┬───────────────────────────────────┘
                              │ CycloneDDS (<1ms)
                              ↓
                    ┌─────────────────────┐
                    │   Go2 機器狗 (MCU)   │
                    │   45+ 動作 / 感測器  │
                    └─────────────────────┘
```

### 2.2 任務分層

| 層級 | 延遲 | 負責任務 | 工具 |
|------|------|----------|------|
| **快系統 (本地)** | <200ms | 避障、導航、偵測、TTS、安全 | Nav2, coco, D435 |
| **慢系統 (雲端)** | 1-5s | 語意理解、任務規劃、VLM | Qwen, LLaVA, Whisper |
| **執行層 (Go2)** | 即時 | 動作執行、感測器 | 45+ 動作 |

## 4. 技術規格

### 4.1 本地運算 (Mac VM)

| 組件 | 規格 |
|------|------|
| **導航** | Nav2 + slam_toolbox |
| **偵測** | FasterRCNN_MobileNet_V3 (80 類) |
| **深度** | RealSense D435 (0.2-10m) |
| **TTS** | ElevenLabs (eleven_turbo_v2_5) |
| **安全** | MAX_LINEAR=0.3m/s, MAX_ANGULAR=0.5rad/s |

### 4.2 雲端運算 (GPU Server)

| 項目 | 規格 |
|------|------|
| **GPU** | NVIDIA RTX PRO 6000 Blackwell (~48GB VRAM) |
| **LLM** | Qwen2.5-72B-Instruct |
| **VLM** | LLaVA-1.6-34B |
| **STT** | Whisper Large V3 |
| **追蹤** | SAM 2 |

### 4.3 連接規格

| 模式 | IP | 延遲 | 備註 |
|------|-----|------|------|
| CycloneDDS | 192.168.123.161 (MCU) | <1ms | SLAM/Nav2 首選（LiDAR 高頻） |
| WebRTC | 192.168.12.1 / 192.168.123.161 | ~50-100ms | 影像 + 控制可用，但 LiDAR 低頻 |

## 5. 常用指令速查

```bash
# 環境設定
source /opt/ros/humble/setup.bash && source install/setup.bash

# 有線模式啟動
export CONN_TYPE=cyclonedds && export ROBOT_IP=192.168.123.161
ros2 launch go2_robot_sdk robot.launch.py slam:=true nav2:=true

# 物件偵測
ros2 run coco_detector coco_detector_node --ros-args -p device:=cpu

# 執行動作
ros2 topic pub /webrtc_req go2_interfaces/msg/WebRtcReq \
  "{api_id: 1016, topic: 'rt/api/sport/request'}" --once

# TTS
ros2 topic pub /tts std_msgs/String "data: '找到椅子了'"

# 緊急停止
ros2 service call /stop_movement std_srvs/srv/Trigger
```
