## I. 專案願景與核心價值

### 1. 願景

打造一隻「**聽得懂人話、看得見環境、找得到東西**」的 AI 四足機器狗，結合 **Edge AI 即時控制** 與 **雲端大模型高階推理**，實現真正智能的具身互動體驗。

### 2. 核心理念：PawAI

> **「不以衝擊大廠為目標，而是做大廠不敢做的事」**

這不只是一個會移動的工具，而是一隻能：
- **聽得懂人話：** 透過本地 LLM 理解自然語言指令
- **看得懂環境：** 結合視覺辨識與深度感知
- **找得到東西：** 自主導航 + 物件追蹤
- **做得出反應：** 45+ 動作庫 + 語音回饋

### 3. 技術定位

| 項目 | 6 年前方案 | PawAI 方案 |
|------|-----------|-----------| 
| 機器人 | TurtleBot3 (兩輪) | **Go2 四足機器狗** |
| 架構 | 全雲端 AWS | **雙層：本地 + 雲端** |
| 避障 | 雲端決策 | **本地 Nav2 + LiDAR** |
| AI 模型 | 小模型 + SageMaker | **70B LLM + VLM** |
| 延遲 | 2-5 秒 | **即時 <200ms** |
| 離線 | ❌ | ✅ |
| 通訊協定 | AWS IoT | **MCP (業界前沿)** |

## II. 雙層 AI 系統架構

### 1. 架構概覽

```
                    ┌────────────────────────────────────┐
                    │  學校 GPU Server (RTX 6000 Blackwell) │
                    │  ┌─────────────────────────────┐   │
                    │  │ 慢系統 (System 2) - 高階推理 │   │
                    │  ├─────────────────────────────┤   │
                    │  │ • 本地 LLM (Qwen-72B)       │   │
                    │  │ • VLM (LLaVA-34B)           │   │
                    │  │ • Whisper Large V3 (STT)    │   │
                    │  │ • SAM 2 (物件追蹤)          │   │
                    │  └─────────────────────────────┘   │
                    └──────────────┬─────────────────────┘
                                   │ HTTP API (1-5s)
                                   ↓
┌──────────────────────────────────────────────────────────────┐
│                    Mac VM (Ubuntu 22.04) — 大腦              │
│  ┌─────────────────────────────────────────────────────┐    │
│  │           快系統 (System 1) - 即時控制 <200ms        │    │
│  ├─────────────────────────────────────────────────────┤    │
│  │ • Nav2 + SLAM (避障導航)                             │    │
│  │ • coco_detector (本地物件偵測)                       │    │
│  │ • 深度攝影機 (RealSense D435)                        │    │
│  │ • TTS 語音回饋 (ElevenLabs)                          │    │
│  │ • Safety Layer (速度限制/緊急停止)                   │    │
│  │ • Skills 模組化封裝                                  │    │
│  │ • MCP 協定介面                                       │    │
│  └─────────────────────────────────────────────────────┘    │
└──────────────────────────────────────────────────────────────┘
                               │ CycloneDDS (<1ms)
                               ↓
                    ┌─────────────────────┐
                    │      Go2 機器狗      │
                    │   (MCU 192.168.123.161)  │
                    │   45+ 動作 / LiDAR / Camera │
                    └─────────────────────┘
```

### 2. 任務分層

| 層級 | 位置 | 延遲 | 負責任務 |
|------|------|------|----------|
| **快系統** | Mac VM 本地 | <200ms | 避障、導航、偵測、TTS、安全控制 |
| **慢系統** | GPU Server | 1-5s | 語意理解、任務規劃、VLM、STT |
| **執行層** | Go2 MCU | 即時 | 動作執行、感測器資料 |

### 3. 關鍵技術組件

#### 本地運算 (Mac VM)

| 組件 | 技術 | 用途 |
|------|------|------|
| **導航** | Nav2 + slam_toolbox | 自主避障導航 |
| **偵測** | coco_detector (MobileNet) | 80 類 COCO 物件偵測 |
| **深度** | Intel RealSense D435 | 真實距離估計 |
| **語音** | ElevenLabs TTS | 語音回饋 |
| **安全** | Safety Layer | 速度限制 (0.3m/s) |
| **介面** | MCP over rosbridge | LLM 控制介面 |

#### 雲端運算 (GPU Server)

| 組件 | 模型 | 用途 | 推論時間 |
|------|------|------|----------|
| **語意** | Qwen2.5-72B | 指令理解、任務規劃 | ~2s |
| **視覺** | LLaVA-34B | 複雜場景理解 | ~3s |
| **語音** | Whisper Large V3 | 語音轉文字 | ~1s |
| **追蹤** | SAM 2 | 互動式物件追蹤 | ~500ms |
