                    └──────┬──────┘

                    │ GPU Server  │
                    │ YOLO + DA3  │
                    └─────────────┘

**重點說明（口頭）：**
1. 使用者說「找水瓶」
2. LLM 理解意圖，呼叫 MCP 工具
3. GPU Server 分析影像，回傳 JSON（不是圖片！）
4. Go2 根據結果移動、找到目標
