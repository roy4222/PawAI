| `s2.png` | 0.6 m | 1.16 m | 0.52 |
| `1m.png` | 1.0 m | 1.51 m | 0.66 |
| **平均** | | | **0.60** |

### 校正結果

```python

SCALE_FACTOR = 0.60  # Go2 廣角鏡頭校正
```

## 🔄 Task 5: 端到端整合測試（進行中）

**目標：** 驗證完整感知閉環：Go2 拍照 → DA3 深度估計 → LLM 避障決策 → Go2 執行

### 測試流程（Windows 中轉版）

```mermaid
sequenceDiagram
    participant Go2 as Go2 機器狗
    participant Mac as Mac VM
    participant Win as Windows
    participant GPU as GPU Server
    participant LLM as LLM (Kilo Code)
    
    LLM->>Mac: 呼叫 /capture_snapshot
    Mac->>Go2: 拍照
    Go2-->>Mac: 圖片存至 /tmp/snapshot_latest.jpg
    Mac->>Win: SCP 傳送圖片
    Win->>GPU: SCP 傳送圖片
    GPU->>GPU: DA3 推論
    GPU-->>Win: 回傳避障建議 JSON
    Win-->>Mac: 手動告知結果
    LLM->>LLM: 解析建議，決定動作
    LLM->>Mac: 呼叫 /move_for_duration
    Mac->>Go2: 執行移動
```

### 測試步驟

**Step 1: 準備場景**
- 放置障礙物在 Go2 前方約 0.5-1m
- 確認 GPU Server 已啟動 (`uvicorn ... --port 8050`)

**Step 2: Go2 拍照**
```bash
# Mac VM
ros2 service call /capture_snapshot std_srvs/srv/Trigger
```

**Step 3: Windows 中轉**
```powershell
# Windows PowerShell
# 從 Mac VM 下載
scp roy422@192.168.1.200:/tmp/snapshot_latest.jpg C:\Users\你的帳號\test.jpg

# 上傳到 GPU Server
scp -P 8022 C:\Users\你的帳號\test.jpg roy422@140.136.155.5:~/test.jpg

# 在 GPU Server 測試
ssh roy422@140.136.155.5 -p 8022
curl -X POST http://localhost:8050/perceive -F "image=@~/test.jpg"
```

**Step 4: 驗證避障建議**

| 場景 | 預期回應 |
|------|---------|
| 前方 < 1m 有障礙 | `⚠️ 建議向 X 繞行` |
| 前方暢通 | `✅ 前方暢通，可安全前進` |
| 三面受阻 | `🛑 建議後退或停止` |

**Step 5: LLM 根據建議控制 Go2**
- 若建議繞行 → `/move_for_duration` 轉向
- 若前方暢通 → `/move_for_duration` 前進

## ⚠️ 技術筆記

### Perception Server 架構

```mermaid
graph LR
    subgraph GPU["GPU 伺服器 :8050"]
        FastAPI[FastAPI]
        DA3[DA3 ViT-L]
        Analyze[區域分析]
    end
    
    Client[圖片上傳] --> FastAPI
    FastAPI --> DA3
    DA3 --> Analyze
    Analyze --> Response[JSON 回應]
```

### DA3 區域分析邏輯

```python
# 分割為左中右三區（各佔畫面 1/3）
left_region = depth_map[:, :w//3]
center_region = depth_map[:, w//3:2*w//3]
right_region = depth_map[:, 2*w//3:]

# 中央下半部 = 前方最近障礙物
center_lower = depth_map[h//2:, w//3:2*w//3]
front_obstacle = np.percentile(center_lower, 25)  # 取較近距離
```

### 避障決策邏輯

| 條件 | 建議 |
|------|------|
| `front > 1.0m` | ✅ 前方暢通，可前進 |
| `front < 1.0m && right > left` | ⚠️ 建議向右繞行 |
| `front < 1.0m && left > right` | ⚠️ 建議向左繞行 |
| 三面受阻 | 🛑 建議後退或停止 |
