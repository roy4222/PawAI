## 🏗️ 技術架構

### 感知融合流程

```
使用者：「幫我找水瓶」
      ↓
LLM 呼叫 find_object("水瓶") MCP Tool
      ↓
HTTP POST → GPU Server :8001/find_object
      ├── YOLO-World：偵測物體 → bbox, center (42ms)
      └── DA3 Metric：深度估計 → distance (289ms)
            ↓ 融合輸出 JSON
{ "label": "水瓶", "distance_m": 0.5, "direction": "正前方", "cmd_vel": {...} }
      ↓
Go2 機器狗執行移動
```

### 導航指令邏輯

| 物件位置 | cx 範圍 | angular_z | 說明 |
|----------|---------|-----------|------|
| 左側 | < 213 | +0.3 | 向左轉 |
| 正前方 | 213-427 | 0.0 | 直行 |
| 右側 | > 427 | -0.3 | 向右轉 |

| 距離 | linear_x | 說明 |
|------|----------|------|
| < 0.3m | 0.0 | 停止 |
| 0.3-1.0m | 0.1 | 慢速 |
| > 1.0m | 0.2 | 正常 |

## 🛠️ 如何啟動服務

### 啟動 YOLO Server (Port 8000)

```bash
cd ~/Yoloworld
source .venv/bin/activate
uvicorn YoloworldGPU:app --host 0.0.0.0 --port 8000
```

### 啟動 Perception Server V2 (Port 8001) ⭐

```bash
source ~/miniconda3/bin/activate depth-v2
cd ~/Yoloworld  # 或 ~/Depth_Anything_V2/.../perception_server
uvicorn perception_server_v2:app --host 0.0.0.0 --port 8001
```

### API 測試指令

```bash
# 找水瓶
curl -X POST "http://localhost:8001/find_object/summary" \
  -F "image=@test_image.jpg" \
  -F "target=bottle"

# 找人
curl -X POST "http://localhost:8001/find_object" \
  -F "image=@test_person.jpg" \
  -F "target=person"
```
