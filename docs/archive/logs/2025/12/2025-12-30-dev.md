## 🔌 Go2 連線機制理解

今天深入研究了 `go2_robot_sdk` 如何連接機器狗：

### WebRTC 連線流程

```
go2_driver_node
      ↓
WebRTCAdapter (管理所有連線)
      ↓
Go2Connection (單一機器狗連線)
      ↓ WebRTC (UDP/ICE)
Go2 機器狗 (192.168.12.1)
```

### 連線步驟（go2_connection.py）

1. 建立 WebRTC Offer
2. HTTP 請求取得機器狗公鑰 (RSA)
3. 用公鑰加密 SDP Offer → 發送給機器狗
4. 機器狗回傳加密的 SDP Answer → 解密
5. 設定 Remote Description
6. ICE 候選交換 → UDP 通道建立
7. SCTP Data Channel 開啟 ✅

### 關鍵檔案

- `go2_driver_node.py`：ROS2 節點入口
- `webrtc_adapter.py`：WebRTC 連線管理
- `go2_connection.py`：WebRTC 握手實作
