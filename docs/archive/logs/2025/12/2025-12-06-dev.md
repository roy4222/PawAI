## ⚠️ 重要：VM 網路環境說明（來自 2025/12/01 除錯經驗）

你的 VM 有**雙網卡**，必須正確配置 CycloneDDS：

| 網卡 | IP | 用途 |
|------|-----|------|
| `lo` | 127.0.0.1 | **本機 ROS2 通訊（必須用這個！）** |
| `enp0s1` | 192.168.12.222 | 連接 Go2 機器狗 |
| `enp0s2` | 192.168.1.125 | 連接家用網路（Windows 在這裡） |

**關鍵設定：** 使用 `/home/roy422/local_only_v2.xml` 強制本機通訊走 loopback！

```bash
# 每個 Terminal 都必須設定！
export RMW_IMPLEMENTATION=rmw_cyclonedds_cpp
export CYCLONEDDS_URI=/home/roy422/local_only_v2.xml
```

### 4.3 驗證 rosbridge 能看到 Go2 Topics ✅

```python
python3 << 'EOF'
import asyncio, websockets, json
async def test():
    async with websockets.connect('ws://127.0.0.1:9090') as ws:
        await ws.send(json.dumps({'op': 'call_service', 'service': '/rosapi/topics'}))
        result = json.loads(await ws.recv())
        topics = result.get('values', {}).get('topics', [])
        print(f"✅ rosbridge 看到 {len(topics)} 個 topics")
asyncio.run(test())
EOF
```

**結果：** ✅ rosbridge 看到 **25 個 topics**：
- `/camera/camera_info`
- `/camera/image_raw`
- `/cmd_vel`
- `/go2_states`
- `/imu`
- `/odom`
- `/scan`
- `/point_cloud2`
- `/tf`, `/tf_static`
- ... 等

### 4.4 Kilo Code MCP 整合測試 ✅

#### 4.4.1 配置 Kilo Code

建立 `.kilocode/mcp.json`：

```json
{
  "mcpServers": {
    "ros-mcp-server": {
      "command": "uvx",
      "args": ["ros-mcp", "--transport", "stdio"],
      "env": {
        "ROS_BRIDGE_URL": "ws://127.0.0.1:9090"
      },
      "disabled": false,
      "alwaysAllow": []
    }
  }
}
```

#### 4.4.2 建立 ROS2專家 Mode

在 Kilo Code 中建立自訂 Mode：
- **Name:** ROS2專家
- **Slug:** ros2
- **Role Definition:** 包含 MCP 工具使用說明與安全規則

## ⚠️ 已知問題與解決方案

| 問題 | 原因 | 解決方案 |
|------|------|---------|
| **Terminal 間看不到 topics** | CycloneDDS 走錯網卡 | 確保所有 Terminal 都設定 `CYCLONEDDS_URI=/home/roy422/local_only_v2.xml` |
| **ros2 node list 可看到，但 topic hz 無數據** | Discovery 成功但 Data Transfer 失敗 | 使用 loopback 配置 |
| **Port 9090 無法從 Windows 存取** | 防火牆或網路配置 | 確認 enp0s2 (192.168.1.125) 可 ping 通 |
| **uvx 找不到** | 未載入環境 | `source ~/.bashrc` 或重開 Terminal |
| **publish_for_durations JSON 錯誤** | ros-mcp-server API 參數格式 | 改用 `publish_once()` 多次呼叫 |
| **影像無法顯示** | /camera/image_raw 是原始二進位 | 需開發 snapshot_service (W7 任務) |

## 🚀 MCP 模式快速啟動（phase1_test.sh）

已將 rosbridge 和 MCP 模式整合到 `phase1_test.sh`，使用更方便：

### 新增指令

| 指令 | 用途 |
|------|------|
| `zsh phase1_test.sh bridge` | 啟動 rosbridge (Port 9090) |
| `zsh phase1_test.sh mcp` | 啟動 Go2 Driver 精簡版（不含 SLAM/Nav2） |

### 啟動順序

```bash
# Terminal 1：環境檢查（只需執行一次）
zsh phase1_test.sh env

# Terminal 2：啟動 rosbridge
zsh phase1_test.sh bridge

# Terminal 3：啟動 Go2 Driver
zsh phase1_test.sh mcp

# 然後在 Kilo Code 中切換到「ROS2專家」Mode
```

> ⚠️ **注意：** 如果 rosbridge 已經在跑（Address already in use），跳過 `bridge` 步驟即可。
