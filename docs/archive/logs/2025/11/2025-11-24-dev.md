{

  "object": "cup",
  "bbox": [x, y, width, height],  // 左上角 + 寬高
  "center": [u, v],               // 框框中心點 (左上角為原點)

*   原點: 影像左上角 (0, 0)
*   u軸: 向右為正
*   v軸: 向下為正
*   center = [x + width/2, y + height/2]

#### 座標轉換 Level 1 (地面假設法) 預習
```python
# 核心概念
# 1. 從 C組收到 2D 像素座標 [u, v]
# 2. 從 /camera/camera_info 取得內參矩陣 K
# 3. 假設物體在地面 (Z = 0)
# 4. 計算相機射線與地面交點
# 5. 使用 tf2 轉換到 map frame
# 6. 發布 PoseStamped 到 /target_pose

# 待實作流程
input: center [u, v]  (from C組)
  ↓
取得內參矩陣 K
  ↓
計算射線方向
  ↓
與 Z=0 平面交點 → [x, y, 0] (base_link frame)
  ↓
tf2 轉換 → [X, Y, 0] (map frame)
  ↓
output: PoseStamped (world frame)
```
