## 🔧 技術問題排除

### 問題 1：GPU Server Port 被佔用

**症狀：**
```
ERROR: [Errno 98] error while attempting to bind on address ('0.0.0.0', 8001): address already in use
```

**解法：**
```bash
lsof -i :8001  # 找 PID
kill -9 <PID>
python perception_server_v2.py
```

### 問題 2：避障後轉回來撞障礙

**原因：** LLM 避障後轉向目標，沒有再次確認前方安全

**解法：** 更新 System Prompt 加入「前進前必檢」規則

```
❌ 錯誤流程：避障 → 轉向目標 → 直接前進 → 撞到障礙！
✅ 正確流程：避障 → 轉向目標 → find_object 確認 → 才前進
```
