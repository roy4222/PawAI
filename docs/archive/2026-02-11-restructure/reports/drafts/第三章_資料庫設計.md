### 3.1.5 關聯說明

| 關聯 | 類型 | 說明 |
|------|------|------|
| tbl_user → tbl_search_log | 1:N | 一個使用者可有多筆搜尋記錄 |
| tbl_user → tbl_action_log | 1:N | 一個使用者可有多筆動作記錄 |
| tbl_object → tbl_search_log | 1:N | 一個物件可被多次搜尋 |
| tbl_object → tbl_location_memory | 1:N | 一個物件可有多個記憶位置 |
| tbl_action → tbl_action_log | 1:N | 一個動作可被多次執行 |

---

### 3.1.6 正規化說明

本設計已達 **第三正規化 (3NF)**：

| 正規化 | 說明 | 符合 |
|--------|------|------|
| 1NF | 所有欄位為原子值，無重複群組 | ✅ |
| 2NF | 所有非鍵欄位完全依賴於主鍵 | ✅ |
| 3NF | 所有非鍵欄位不存在傳遞依賴 | ✅ |

**設計決策**：
- `tbl_object` 獨立存放物件定義，避免在 `tbl_search_log` 中重複儲存物件名稱
- `tbl_action` 獨立存放動作定義，避免在 `tbl_action_log` 中重複儲存 API ID
- `tbl_location_memory` 與 `tbl_object` 分離，因一個物件可能在多個位置被發現

### 3.1.7 SQL 建表語句

```sql
-- T01: 使用者資料
CREATE TABLE tbl_user (
    user_id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_name VARCHAR(50) NOT NULL,
    user_role VARCHAR(20) NOT NULL DEFAULT 'elder',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- T02: 物件類別定義
CREATE TABLE tbl_object (
    object_id INTEGER PRIMARY KEY AUTOINCREMENT,
    object_name_en VARCHAR(50) NOT NULL,
    object_name_zh VARCHAR(50) NOT NULL,
    object_category VARCHAR(30) DEFAULT 'daily',
    is_active BOOLEAN DEFAULT 1
);

-- T03: 尋物搜尋記錄
CREATE TABLE tbl_search_log (
    search_id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id INTEGER NOT NULL,
    object_id INTEGER,
    search_query VARCHAR(200) NOT NULL,
    is_found BOOLEAN DEFAULT 0,
    distance_m REAL,
    direction VARCHAR(20),
    confidence REAL,
    latency_ms INTEGER,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES tbl_user(user_id),
    FOREIGN KEY (object_id) REFERENCES tbl_object(object_id)
);

-- T04: Go2 動作定義
CREATE TABLE tbl_action (
    action_id INTEGER PRIMARY KEY AUTOINCREMENT,
    action_name VARCHAR(30) NOT NULL,
    api_id INTEGER NOT NULL,
    action_category VARCHAR(20) DEFAULT 'safe',
    duration_ms INTEGER DEFAULT 3000,
    is_active BOOLEAN DEFAULT 1
);

-- T05: 動作執行記錄
CREATE TABLE tbl_action_log (
    log_id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id INTEGER NOT NULL,
    action_id INTEGER NOT NULL,
    is_success BOOLEAN DEFAULT 1,
    execution_time_ms INTEGER,
    error_message VARCHAR(200),
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES tbl_user(user_id),
    FOREIGN KEY (action_id) REFERENCES tbl_action(action_id)
);

-- T06: 物品位置記憶
CREATE TABLE tbl_location_memory (
    memory_id INTEGER PRIMARY KEY AUTOINCREMENT,
    object_id INTEGER NOT NULL,
    location_desc VARCHAR(100),
    pos_x REAL,
    pos_y REAL,
    found_count INTEGER DEFAULT 1,
    last_found_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (object_id) REFERENCES tbl_object(object_id)
);
```

### 3.1.9 欄位命名規則

| 類別 | 命名規則 | 範例 |
|------|---------|------|
| 資料表名稱 | `tbl_` + 小寫 + 底線 | `tbl_user`, `tbl_search_log` |
| 主鍵欄位 | 表名單數 + `_id` | `user_id`, `action_id` |
| 外鍵欄位 | 參照表名單數 + `_id` | `user_id`, `object_id` |
| 布林欄位 | `is_` 前綴 | `is_found`, `is_active`, `is_success` |
| 時間欄位 | `_at` 後綴 | `created_at`, `updated_at`, `last_found_at` |
| 描述欄位 | `_desc` 後綴 | `location_desc` |
| 計數欄位 | `_count` 後綴 | `found_count` |
