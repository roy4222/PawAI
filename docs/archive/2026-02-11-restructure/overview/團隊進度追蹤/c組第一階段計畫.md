- ❌ **座標計算**（像素座標 → 世界座標）→ LLM Vision 直接判斷位置
- ❌ **JSON API 格式約定**（複雜的 bbox/center 格式）→ 只需標註圖片
- ❌ **與 A 組確認座標格式**（原 Phase 1 任務）→ 改為確認圖片格式

### 簡化的原因
> **原計畫：** YOLO 回傳 (u, v) → Roy 計算座標轉換 → 世界座標 (x, y, z)
>
> **MCP 新計畫：** YOLO 標註圖片 → MCP 讀取 → **LLM Vision 直接判斷「往左 2 公尺」**
