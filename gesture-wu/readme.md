### 1. 核心觀念：手勢作為「語音狀態機」的切換開關

我們不讓狗狗移動，但讓它「聽」跟「說」的邏輯由你控制。

| 手勢 (Gesture)            | 對應功能模式                     | 陪伴互動細節 (情境說明)                                                                                                                            |
| ------------------------- | -------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------- |
| **食指指引 (☝️)** | **主動聆聽模式 (Listen)**  | **長者想要聊天了** 。手一指，狗狗發出一個輕微的「叮」聲並歪頭（表示正在聽），此時 ASR 開始收音。這解決了喚醒詞失效的問題。                   |
| **握拳 (👊)**       | **立即靜音模式 (Mute)**    | **狗狗太吵了** 。當狗狗正在朗讀報紙、播報提醒或講故事時，長者手一握，狗狗立刻停止說話並進入待命。這比用喊的「閉嘴」更優雅且有效。            |
| **比讚 (👍)**       | **情感回饋模式 (Happy)**   | **狗狗講得很好** 。例如狗狗說了一個笑話，長者比讚，狗狗會發出開心的音效，並在資料庫記錄長者的喜好（強化 AI 的個性化）。                      |
| **OK 手勢 (👌)**    | **指令確認模式 (Confirm)** | **取代口頭回答「正確」** 。例如狗狗問：「張爺爺，現在要幫你打電話給兒子嗎？」長者只需比 OK，狗狗就撥號。這對口齒不清或懶得說話的長者很友善。 |
| **掌心朝前 (🖐️)** | **暫停互動模式 (Pause)**   | **現在想一個人靜靜** 。狗狗關閉所有主動對話功能（但保留跌倒偵測等安全監控），直到下次被喚醒。                                                |

### 2. 進階互動設計（不用大範圍移動，但有動作感）

雖然不導航，但我們可以讓狗狗在原地利用「關節動作」來回應語音：

* **視覺追蹤 (Head Tracking)** ：當開啟「聆聽模式 (☝️)」後，你可以結合人臉辨識，讓狗狗的頭一直對著使用者的臉。雖然它不走路，但長者會覺得這隻狗「真的在聽我說話」。
* **動作輔助語音** ：
* 說「好喔」的時候同時 **點頭** 。
* 說「我不知道耶」的時候同時**歪頭**或 **後退一小步** （原地動作）。
* 當長者太久沒說話時，狗狗可以做一個「伸懶腰」的動作，然後主動問一句：「你要聽首歌嗎？」
